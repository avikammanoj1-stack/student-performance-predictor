# Student Performance Predictor

## Overview
This project predicts student final scores using Machine Learning.

## Technologies Used
- Python
- Pandas
- Scikit-Learn

## Features
- Reads student data from CSV
- Trains a Linear Regression model
- Predicts final scores

## Dataset Columns
- StudyHours
- Attendance
- PreviousMarks
- Assignments
- FinalScore
